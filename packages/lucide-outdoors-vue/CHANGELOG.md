# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.2] - 2026-06-18

### Changed

- No user-facing changes. Released via the automated GitHub Actions OIDC pipeline, with provenance.

## [0.1.0] - 2026-03-20

### Added

- Initial release with 21 Vue components wrapping `lucide-outdoors` icon data
- Trail activity components: `Hiking`, `Walking`, `Running`, `Biking`, `RoadCycling`, `GravelBiking`, `MountainBiking`, `ERoadCycling`, `EGravelBiking`, `EMountainBiking`, `Skiing`, `Snowboarding`
- POI components: `Mountain`, `Water`, `Church`, `TrailSign`, `Eye`, `Home`, `Waterfall`, `Cave`, `Monument`
- Built on `lucide-vue-next`'s `createLucideIcon` -- all standard Lucide props work
- ESM and CJS builds with TypeScript declarations
