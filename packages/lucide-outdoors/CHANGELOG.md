# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-06-18

### Changed

- No user-facing changes. First release published automatically via GitHub Actions using npm OIDC trusted publishing, with provenance.

## [0.1.0] - 2026-03-20

### Added

- Initial release with 21 icons
- Trail activity icons: `hiking`, `walking`, `running`, `biking`, `road-cycling`, `gravel-biking`, `mountain-biking`, `e-road-cycling`, `e-gravel-biking`, `e-mountain-biking`, `skiing`, `snowboarding`
- POI icons: `mountain`, `water`, `church`, `trail-sign`, `eye`, `home`, `waterfall`, `cave`, `monument`
- ESM and CJS builds with TypeScript declarations
- `IconNode` format compatible with `lucide-vue-next`
